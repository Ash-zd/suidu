### 流程说明
现在item里面定义要爬的东西
如下
```python
class GithubTrendingItem(Item):
    # 链接
    url = Field()
    # 仓库名
    repo = Field()
    # 项目描述
    desc = Field()
    # 语言类型
    type = Field()
    # star数
    stars = Field()
    # 今日star数
    today_stars = Field()
    # 所有者
    owner = Field()
    # fork数
    forks = Field()
```


本地写完之后直接用scrapyd-deploy提交到服务器上就行。
