# 前端

## 目录说明
```
├── dist              # 生成的资源目录
└── vue
    ├── code          # 源码目录
    |   ├── build     # webpack构筑文件
    |   ├── config    # webpack配置
    |   └── src       # vue相关文件在这里
    |
    ├── node_modules  # 依赖包
    └── package.json
```

## 代码编写
一般在`vue/code/src` 里面写完直接 `npm run build` 代码会生成到dist目录里面。

## 生产与测试环境说明
api定义在 `vue/code/src/api.js`里
本地使用的话吧服务器那个ip改了就行。
或者改hosts
```
let base = 'http://127.0.0.1';       // 测试环境使用
let base = 'http://123.206.113.243'; // 生产环境使用
```
